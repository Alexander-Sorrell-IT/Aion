# git-tools

Git workflow toolkit. Three skills covering the most-friction operations.

| Component | Triggered by |
|---|---|
| `commit-craft` skill | "commit", "write a commit message", staged changes pending |
| `conflict-resolution` skill | "merge conflict", `<<<<<<<` markers, failed rebase |
| `branch-hygiene` skill | "clean up branches", "what branches do I have" |
| `/clean-gone` command | Prune `[gone]` branches after confirmation |
